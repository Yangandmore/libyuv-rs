libyuv-rs
---

依据libyuv，来完成在rust中的迁移工作。

将提供：
* 基于rust语言提供了基础的yuv处理功能。
* 基于rust中标准库提供的arch SIMD指令集，完成yuv的高密集型计算。提供数倍处理速度。

***注意：
由于SIMD指令集依赖rust中的标准库，因此部分指令集无法使用!!!***
